# Demo 01 製作筆記(對應模組 1、2、3)

## 這頁是用什麼 brief 生出來的(模組 1)

```text
【活動】山霧咖啡開幕週,6/20-6/26,全品項買一送一
【客群】25-40 歲上班族,重視質感與儀式感
【區塊】滿版 hero、倒數計時、三張亮點卡、時間地點交通、注意事項、置底浮動 CTA
【風格】沉穩、霧感、留白多;深墨綠 + 奶油白 + 銅棕點綴;標題用襯線字
【CTA】「加入 LINE 領開幕券」(demo 連到 example.com)
加上 OG meta tags;手機優先;繁體中文;文案不要 AI 樣板腔
```

## 視覺規格卡(模組 2,hero 若要換成 AI 生成圖時用)

- 主體:山嵐間的咖啡店窗景,一杯熱拿鐵在窗台,蒸氣上升
- 風格:日系極簡商業攝影,底片顆粒,安靜高級
- 光線:清晨霧中柔光
- 構圖:16:9,主體置右 1/3,左側留白疊標題
- 配色:深墨綠、奶油白、木質棕,低飽和
- 負面:無文字、無人臉、無過度 HDR

demo 版為了零依賴與離線可開,hero 用「漸層 + 霧氣動畫 + 粒子」代替照片;
實際活動把 hero 背景換成規格卡生出的圖即可(模組 2 的「文字用網頁疊」做法)。

## 特效(模組 3)

- 課堂教學用 tsParticles CDN(講義 prompt 範本);本 demo 內嵌等效的零依賴
  星座連線粒子,參數集中在 `CONF`(密度 / 速度 / 連線距離 / 顏色),手機自動減半。
- 其餘動態:CSS 霧氣漂移、進場 staggered rise、IntersectionObserver 捲動進場、
  置底 CTA bar 捲過一屏才出現。一頁只有 hero 一個搶眼特效,符合「一頁一特效」原則。
