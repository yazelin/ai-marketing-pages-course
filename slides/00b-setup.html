MODULE 0

行前準備:
工具與心態

要準備的三樣東西(全免費)

  • AI 聊天工具 — ChatGPT / Claude / Gemini 任選 付費版生長網頁較不會截斷
  • GitHub 帳號 — 免費網頁主機 + 所有活動頁的保險箱
  • 純文字編輯器 — 記事本就行;裝 VS Code 也不用學

不需要:程式語言、命令列、付費軟體。

檔案習慣,先養成

  • 一個活動一個資料夾,網頁存 index.html
  • 存檔編碼一律 UTF-8,否則中文變亂碼
  • 檔名英文小寫加連字號:hero-1920.jpg 會變成網址的一部分
  • 重要版本先另存 index-v2.html 再改,隨時有路可退

這門課最重要的一句話

你不是在學寫程式,
你是在學驗收。

你是總監,AI 是工讀生

  • 工讀生動作快、不喊累,但會自作主張、記錯話、謊稱做完
  • 總監的工作:brief 清楚、一次只下一兩個修改指令、逐項驗收
  • 出問題不用自己修:把錯誤訊息原封不動丟回去
  • 每個模組的「常見坑」就是你的驗收清單

30 秒測試:你準備好了沒
