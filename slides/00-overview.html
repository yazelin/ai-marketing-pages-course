AI 互動行銷頁實作課

不寫程式,做出會動、會玩、能上線的活動網頁。

這門課給誰上

  • 會企劃活動、會寫文案的行銷人
  • 天天用 ChatGPT / Claude / Gemini 聊天,但不會寫程式
  • 每次要做活動頁,都得排隊等工程師或外包
  • 需要的只有:瀏覽器、一個 AI 聊天工具、免費 GitHub 帳號

核心轉換

把你已經有的「會描述需求」,變成「能自己做出活動頁」。

課程方法論:甲方思維

  • AI 時代你升格成甲方——AI 是史上最好說話的乙方:重做不加價、半夜交件不擺臉色
  • 但爛甲方拿到爛交付物:規格寫多清楚,交付物就多接近你要的
  • 兩件事不能外包:開規格(模組 S)、做驗收(模組 8);施工交給 AI
  • 三步合稱「AI 發包法」:開規格 → 下發包 → 做驗收

程式碼是耗材,AI 隨時能重生一份;規格和驗收的判斷力才是你帶走的東西。

完課後你能做到

  • 用 AI 對話生出一整頁有質感的活動網頁
  • 用結構化 prompt 生出風格一致的活動視覺
  • 幫頁面加上粒子、蒸氣、動態背景等互動特效
  • 做出拉霸抽獎、心理測驗等互動遊戲
  • 自己上線(GitHub Pages),看懂何時要 Vercel / Render / Cloudflare

課程三原則

  • 零 build — 全部是單一 HTML 檔,雙擊就開,不裝開發工具
  • 免費 tier — 整條生產線不花一毛錢
  • prompt 是核心技能 — 帶走的不是程式碼,是跟 AI 要東西的方法
